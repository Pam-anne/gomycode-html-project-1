# Portfolio Web Page Project

## Overview
This repository contains my **Portfolio Web Page** assignment, completed as part of the GomyCode Software Development course. The project is designed in **two parts**:

1. **HTML Structure Project** – Creating the semantic structure of the portfolio page.  
2. **CSS Styling Project** – Styling the page using external CSS to enhance the design and usability.  

The portfolio showcases my skills, projects, and contact information in a clean and professional layout.

---

## Project 1: HTML Structure

### Objectives
- Use **semantic HTML tags** to structure the webpage.
- Add a **navbar** with links to: Home, About, Work, Resume, Contact.
- Create the following sections:
  - **Home** – Introduces me with a profile picture and a short description.
  - **About** – Describes my background, skills, and interests.
  - **Work** – Lists technologies I use and showcases projects using tables and lists.
  - **Resume** – Provides a link to download my resume and an audio element.
  - **Contact** – A functional HTML form to collect user messages.
- Add a **page footer** with copyright and optional social links.
- Include multimedia elements such as **video** and **audio**.

### Features Implemented
- Profile picture displayed using `<img>` tag.
- Multimedia elements:
  - Background video in the Home section.
  - Audio sample in the Resume section.
- Lists and tables used for technologies and project showcase.
- Functional HTML contact form with required fields.
- Inline CSS used initially to demonstrate basic styling.

---

## Project 2: CSS Styling

### Objectives
- Create an external CSS file `styles.css` and link it to the HTML page.
- Change the **navbar display** to inline using `<ul>` and `<li>`.
- Use the **Roboto font** for all text.
- Add class attributes to HTML elements for more precise styling.
- Make the design **visually engaging** and **user-friendly**.
- Implement **responsive design** for mobile devices.

### Features Implemented
- Sticky navbar that remains visible while scrolling.
- Smooth scrolling for navigation links.
- Background video styled with overlay for text readability.
- Home section made full viewport height (`100vh`) for better visual impact.
- Buttons, links, and form elements have hover and focus effects.
- Sections styled with subtle shadows, rounded corners, and gradient backgrounds.
- Footer includes social media links for professional contact.
